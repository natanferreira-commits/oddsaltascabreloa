# LP Cabreloa · Grupo de Odds Altas

Landing page curta pra captação de leads pro grupo de odds altas do Cabreloa.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- next/font pra Google Fonts (Anton + Inter)

## Rodar local

```bash
npm install
npm run dev
```

## Deploy

Deploy automático via Vercel na branch `main`.

## Link do bot

O botão CTA aponta pra `https://t.me/+txgd2kz5u38xZjYx` (link do grupo).
Trocar em `app/page.tsx` na variável `botLink`.
